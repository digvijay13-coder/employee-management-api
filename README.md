Employee Management API

A RESTful API built with Spring Boot, Java, and JPA for managing employee records. Allows adding, updating, deleting, and retrieving employee information.

Features

Add new employees

Retrieve all employees

Retrieve employee by ID

Update employee details

Delete employees

Project Structure
Spring-boot-6/
├── src/
│   ├── main/java/com/MainApp/
│   │   ├── Controller/   # Handles API endpoints
│   │   ├── Entity/       # Employee model
│   │   ├── Repository/   # Database interactions
│   │   └── Service/      # Business logic
│   ├── main/resources/   # Config files
├── .gitattributes
├── .gitignore
├── mvnw / mvnw.cmd
└── pom.xml

Technologies
Java 17, Spring Boot, Maven, MySQL, JPA, REST API
